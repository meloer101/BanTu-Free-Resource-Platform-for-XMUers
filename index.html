About BanTu

BanTu 是一个由厦门大学学生自发维护的开源学习资源聚合平台。我们致力于收集、整理和分享各类课程资料、复习笔记、考试真题与学习参考资料，帮助每一位同学在求学之路上少走弯路。

本项目所有资源均由社区成员自愿贡献，欢迎任何在校学生提交 Pull Request 上传你珍藏的学习资料。无论是课程 PPT、手写笔记还是往年试卷，都是对学弟学妹们无价的馈赠。

使用说明：你可以直接通过本站检索所需资源，也可以前往我们的 GitHub 仓库浏览完整目录结构。资源按院系与课程分类存放，查找起来十分便捷。

贡献指南：若你有优质资料希望分享，请 Fork 项目仓库，将文件按规范放置在对应目录下，提交 PR 并附上简要说明即可。我们会在审核后合并你的贡献，你的名字也将被记录在贡献者名单中。
